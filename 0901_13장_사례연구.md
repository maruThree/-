## [키워드]
- curl : 커맨드 라인으로 웹브라우저 처럼 활동 할수 있게 해주는 기술
- /var/adm : 유닉스 기반 로그 기록을 담는 디렉토리
- 무명 페이징 : 실행 가능한 상태이긴 하지만 스와핑을 기다리면서 블록된 상태
- netstat : 네트워크 상태를 알 수 있는 명령어
- vmstat : 프로세스, 메모리, 페이징, I/O 블럭, CPU 활동 사항들의 정보를 출력하는 기능
